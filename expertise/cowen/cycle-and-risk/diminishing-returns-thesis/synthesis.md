# Diminishing Returns Thesis

Cowen's thesis that each successive Bitcoin bull cycle delivers a lower percentage return than the prior cycle. Used to set realistic cycle-top targets and ground long-horizon expectations.

## Current Thesis

Cowen holds that the four-year cycle is still intact but delivering diminished returns relative to prior cycles — a pattern he treats as normal maturation rather than cycle death. The current cycle is tracking close to the 2014 analog on ROI-from-yearly-open metrics (0.62 now vs. 0.54 in 2014 at the equivalent stage), which he reads as confirmation of an attenuating but still-functioning cycle. BTC continues to underperform major asset classes on a YTD basis, and Cowen does not expect a within-cycle catch-up; meaningful outperformance is deferred to the next cycle, with the current path projected through a June local low and an October primary low.

## Key Evidence

- BTC is underperforming every major asset class (S&P 500, Nasdaq, gold, energy) on a YTD basis as of May 2026, widening the relative gap versus traditional markets (PnveYgxtd1w).
- BTC's recent price recovery has only returned it to levels from ~February 2026, producing zero net new gains for the year — a hallmark of a cycle with weakening momentum (PnveYgxtd1w).
- Cowen explicitly does not expect a catch-up trade within the current cycle, framing any recovery as a next-cycle event: "perhaps in the next cycle we'll see it go back up" (PnveYgxtd1w).
- At the equivalent stage in the 2014 cycle, BTC's ROI from the yearly open was 0.54; the current reading is 0.62 — close enough that Cowen treats it as confirmation the cycle is tracking within normal variance, not breaking down (CNiGShpj3M0).
- Cowen argues the correct 2014 comparison must use ROI from the yearly open (not from the late-2013 high), because the pre-open crash would otherwise artificially inflate the apparent analog return — supporting a methodologically consistent diminishing-returns read (CNiGShpj3M0).
